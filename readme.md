# Slack Deletron API

Uses Vercel Serverless functions

To run locally, install Vercel CLI globally with `npm i vercel -g`. Next in the root directory, run `vercel dev --listen 5005` and the project will be available on localhost:5005.

Current endpoints include

- `api/auth/redirect.ts`
- `api/user/profile.ts`
- `api/user/userdetails`
